Sobre o Projeto

Este projeto consiste no desenvolvimento de uma página web estática como parte das atividades da disciplina INE5646 - Programação para Web. O objetivo principal é aplicar conceitos fundamentais de desenvolvimento front-end, utilizando HTML5 semântico para a estruturação do conteúdo e CSS3 para estilização e definição do layout.

Objetivos da Prática

Os principais objetivos incluem a familiarização com a estrutura de documentos HTML, o uso correto de tags semânticas e a separação entre conteúdo e estilo.
